Navigation basics

  • Navigate Class : Ctrl + N (⌘+O)
    • 클래스명의 대문자 항목을 줄임말로 검색 가능 (ex. OperationToIterator → OTI로 검색)
  • Navigate File : Ctrl + Shift + N (⌘+⇧+O)
    • class 검색과 동일하게 대문자 줄임말 사용 가능
    • path로 검색가능 (ex. /test/test.java 처럼 경로 입력해서 검색 가능)

Symbol lookup

  • Navigate Symbol : Ctrl + Alt + Shift + N (⌘+⌥+O)
    • class 검색과 동일하게 대문자 줄임말 사용 가능
    • 패키지명(줄임말가능).메소드명의 형태로 검색 가능 (ex. rx.operators.OperationSumTest 클래스 내 존재하는 testEmptySumFloats() 라는 메소드 검색 시 OST.sumfloat 입력만 해도 찾을 수 있음

Find in path

  • Find in path : Ctrl + Shift + F (⌘+⇧+F)
    • preview 화면에서 바로 소스를 볼 수 있음

Search everywhere

  • Search everywhere : Double Shift (Double ⇧)
    • 말 그대로 어떤 것이든 검색함
    • tab으로 다른 섹션으로 이동가능
    • Run Configurations에서 설정해둔 서버 이름으로 검색해서 Debug / Run 으로 서버 구동

Working without mouse

  • 마우스 없이 키보드로 작업하는 것을 추천함 (키보드만으로도 충분히 가능)

Working without editor tabs

  • Project, bug report, maven 등의 탭을 열어두고 쓰지 말아라.
  • 인텔리제이는 탭이 필요없다. 비효율적이다. 쓰지마라.
  • Recent Files : Ctrl + E (⌘+E)
    • 최근사용한 파일을 보여주는 팝업
  • Recently Changed Files : Ctrl + Shift + E (⌘+⇧+E)
    • 최근 변경한 파일을 보여주는 팝업
  • 탭 이동 : Alt + 1, 2, 3, 4 … 0
    • keymap에서 설정한 단축키로 각 탭들을 열고 닫고 가능
    • Alt + 1(기본 설정) 은 Project 탭을 열고 닫기함

Autoscroll from/to source

  • Project와 Source 탭 간의 연동
  • from source인 경우 현재 작업하고 있는 소스 기준으로 Project탭에서 해당 위치로 자동 스크롤해서 보여줌
  • to source인 경우 Project탭에서 보여지고 있는 위치의 소스를 보여주는 듯 (정확하게 확인이 안됨…..) ← 이걸 더 추천한다고 함
  • Jump to Source : F4 (⌘+↓)

Create new file/folder

  • New : Alt + Ins (⌘+N)
    • Project 창에서 실행 시 새로운 항목 생성
    • 디렉토리 생성 시 단계 별로 각각 생성하지말고 /test/blah/blah 이런식으로 한번에 생성

Scratch files

  • Find Action : Ctrl + Shift + A (⌘+⇧+A)
  • Scratch Files : Ctrl + Shift + Alt + Ins (⌘+⇧+N)
    • 테스트용으로 소스를 작성하고 싶은 경우라던가 할 때 임의의 파일을 생성해줌

Hide all windows/panels

  • Hide All Windows : Ctrl + Shift + F12 (⌘+⇧+F12)
    • 모든 윈도우(탭)을 토글한다.
    • 비슷한 기능으로 메뉴 → View → Appearance → Enter Distruction Free Mode가 있고 Enter Presentation Mode 에서는 모든 항목을 크게 보여줌
  • Stretch to Left/Right : Ctrl + Shift + ←/→ (⌘+⇧+←/→)
    • 각 윈도우(탭)의 크기 조정 (editor 창에서는 적용 안됨)

Syntax aware selection

  • Extend Selection : Ctrl + W (⌥+Up)
    • 라인 → 괄호 → 메소드 → 클래스 작은 항목부터 넓게 선택

Move/duplicate/delete lines

  • Move Line Up/Down : Shift + Alt + Up/Down (⌥+⇧+Up/Down)
  • Choose Content to Paste : Ctrl + Shift + V
    • 복사했던 항목들이 저장됨
  • Delete Line : Ctrl + Y (⌘+Backspace)

Multiple cursors

  • Edit Multiple Lines : Alt + Shift + Mouse click
  • Find Next / Move to Next Occurrence : F3 (⌘+G)
  • Add Selection for Next Occurrence : Alt + J (^+G)
  • Select All Occurrence : Shift + Alt + J (^+⇧+G)

Vim plugin(21:10)

  • ideavim plugin 설치

Clean up code

  • Auto-Indent Lines : Alt + Ctrl + I (^+⌥+I)
  • Reformat Code : Alt + Ctrl + L (⌥+⌘+L)
  • Settings - Editor - Code Style 에서 상세 지정 가능
  • 코드 선택 후 Alt + Enter 로 Adjust Code Style에서도 설정 가능

Language injection

  • 변수 선언 후 우변에서 Alt + Enter 로 Inject Language or Reference 선택 > 원하는 언어 형식 선택 (ex. JSON) > 다시 Alt + Enter > Edit JSON Fragment 하면 나오는 편집창에서 입력, RegExp 확인도 가능

Code completion

  • Parameter Info : Ctrl + P (⌘+P) 파라미터가 들어갈 () 괄호 안에서 누르면 정보 보여줌
  • Completion Basic : Ctrl + Space (^+Space)
  • Completion SmartType : Shift + Ctrl + Space (^+⇧+Space)
  • Completion Cyclic Expand Word : Alt + Slash/ (⌥+/) 첫글자 입력 후 단축키 입력 시 해당 문자로 시작하는 단어들의 사이클을 돎
  • Complete Current Statement : Shift + Ctrl + Enter (⇧+⌘+Enter)

Code templates

  • Settings - Editor - Live Template
  • Insert Live Template : Ctrl + J (⌘+J)
  • Surround With : Ctrl + Alt + T (⌥+⌘+T) 감쌀 문장에서 단축키 입력

Postfix completion

  • null 체크 깜빡했을 때 object.notnull 선택하면 자동완성 됨
  • emmet 형식 사용 가능 ex) #myDiv>ul>li*10 + Tab

Advanced navigation: go to supermethod, go to superclass, type hierarchy, method hierarchy

  • Navigate → Declaration : Ctrl + B (⌘+B)
  • Navigate → Super Method : Ctrl + U (⌘+U)
  • Navigate → Implementation : Ctrl + Alt + B (⌥+⌘+B)
  • Navigate → Type Hierarchy : Ctrl + H (^+H)
  • Navigate → Method Hierarchy : Shift + Ctrl + H (⇧+⌘+H)
  • Navigate → Jump to Navigation Bar : Alt + Home (⌘+Up)

Structure popup

  • Navigate → File Structure : Ctrl + F12 (⌘+F12)
  • Structure : Alt + 7 (⌘+7)

Inspections

  • Settings - Editor - Inspections에서 설정 가능
  • Menu - Analyze - Inspect Code

Structural search and replace

  • Structural Search (Action 검색) : try catch 문 등의 구조로 검색 가능
  • Structural Replace (Action 검색) : 해당 구조의 내용 변경도 가능

Analyze data flow

  • Menu - Analyze - Analyze Data Flow to Here 선택한 항목의 값이 어디에서 세팅되고 어떤걸 리턴하는지 알 수 있음
  • Metrics Reloded (plugin) : 모든 종류의 metrics를 계산할 수 있음

Database tools

  • 소스 내에 쿼리문이 있는 경우 Alt + Enter를 통해 직접 쿼리를 실행하거나 컬럼명 변경이 가능함

VCS

  • Version Control - log > Ctrl + D : 비교화면

Debugging

  • Editor 라인 넘버 부분에서 더블클릭하여 breakpoint 설정
  • breakpoint 에서 우클릭 하여 다양한 설정 가능
  • 라인 넘버 부분에서 우클릭 하여 전체를 breakpoint 설정할 수 있고 일부만 설정할 수도 있음. (긴 문장의 람다 식인 경우 부분별로 쪼개서 가능)

Refactoring basics

  • Refactor This : Shift + Ctrl + Alt + T (^+T)
  • Extract → Variable : Ctrl + Alt + V (⌥+⌘+V)

Quicklists

  • Settings - Quicklist 에서 설정 가능
  • VCS Operations Popup : Alt + ` (^+V)
  • Quick Switch Scheme : Ctrl + (^+)

Find action

  • Find Action : Ctrl + Shift + A (⌘+⇧+A)

Productivity guide

  • Menu - Help - Productivity Guide 얼마나 사용했는지 언제 마지막으로 썼는지 등 팝업

Zero latency typing

  • Find Action > Registry > edito.zero.latency 부분에서 체크
